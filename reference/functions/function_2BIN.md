The `->BIN` function converts the string on top of the stack to its binary representation.

## Example ##

{% raw %}
<warp10-warpscript-widget backend="{{backend}}"  exec-endpoint="{{execEndpoint}}">
'42'
->BIN
</warp10-warpscript-widget>
{% endraw %}
